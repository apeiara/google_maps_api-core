# GoogleMapsAPI::Core

Common stuff used by the google_maps_api gem components.

## Installation

Add this line to your application's Gemfile:

    gem 'google_maps_api-core'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install google_maps_api-core
